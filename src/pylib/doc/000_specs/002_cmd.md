# pylib commandline interface
## pylib
```
                                                                               
 Usage: pylib [OPTIONS] COMMAND [ARGS]...                                      
                                                                               
+- Options -------------------------------------------------------------------+
| --version             -v        Show the version of this script             |
| --examples            -x        Shows examples                              |
| --install-completion            Install completion for the current shell.   |
| --show-completion               Show completion for the current shell, to   |
|                                 copy it or customize the installation.      |
| --help                -h        Show this message and exit.                 |
+-----------------------------------------------------------------------------+
+- Commands ------------------------------------------------------------------+
| create-package   Create a package that contains all the boilerplate         |
|                  provided by pylib.                                         |
| extract-lib      Extract the pylib code from the given path.                |
| hello-world      Print hello world. Minimal example of a command.           |
| inject-lib       Inject the pylib code into the given path.                 |
| history          Recall cli history (mainly used for debugging).            |
| dev              contains development subcommands                           |
+-----------------------------------------------------------------------------+

```
### pylib create-package
```
                                                                               
 Usage: pylib create-package [OPTIONS] DST NAME                                
                                                                               
 Create a package that contains all the boilerplate provided by pylib.         
                                                                               
 By default, this function injects a copy of pylib into the package. Pass the  
 -imp flag to import instead.                                                  
                                                                               
+- Arguments -----------------------------------------------------------------+
| *    dst       PATH  Path where the new package shall be created [required] |
| *    name      TEXT  Name of the new package [required]                     |
+-----------------------------------------------------------------------------+
+- Options -------------------------------------------------------------------+
|         -imp        import from pylib instead of inject.                    |
|         -f          pass to skip confirmations                              |
|         -r          run the package via uv after creation                   |
| --help  -h          Show this message and exit.                             |
+-----------------------------------------------------------------------------+

```
### pylib dev
```
                                                                               
 Usage: pylib dev [OPTIONS] COMMAND [ARGS]...                                  
                                                                               
 contains development subcommands                                              
                                                                               
+- Options -------------------------------------------------------------------+
| --help  -h        Show this message and exit.                               |
+-----------------------------------------------------------------------------+
+- Commands ------------------------------------------------------------------+
| install   Install development dependencies. Assumes a local git repo. Will  |
|           not work with uvx.                                                |
| test      Run common test commands. Collection of useful tests, can also be |
|           done via pytest.                                                  |
| lint      Run common linter commands. Collection of useful lints, can also  |
|           be done via ruff.                                                 |
| update    update the pylib code used by this package. Pass the hidden       |
|           --library-update flag to run it automatically.                    |
| mkdoc     create the documentation for the package.                         |
+-----------------------------------------------------------------------------+
```

### pylib history
```
                                                                               
 Usage: pylib history [OPTIONS] [N]                                            
                                                                               
 Recall cli history (mainly used for debugging).                               
                                                                               
+- Arguments -----------------------------------------------------------------+
|   n      [N]  selected history entry [default: -1]                          |
+-----------------------------------------------------------------------------+
+- Options -------------------------------------------------------------------+
| --push    -p        push the selected entry to the top                      |
| --clear   -c        clear (if n not provided, clear all)                    |
| --select  -s        interactively select a history entry and repeat it.     |
| --help    -h        Show this message and exit.                             |
+-----------------------------------------------------------------------------+

```
